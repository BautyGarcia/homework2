# Homework 2

Para compilar y ejecutar cada ejercicio, segui estas instrucciones:

## Requisitos previos

- Compilador g++
- make

## Compilación y ejecución

1. Parate en el directorio del ejercicio que queres compilar. Por ejemplo:

```bash
cd homework2/ej1
```

2. Compila y ejecuta el programa:

```bash
make
```
